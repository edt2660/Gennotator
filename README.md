# Gennotator
Disclaimer: Side project, not officially affiliated with any program
Phage gene autoannotator meant to help with SEA-PHAGES and other PhagesDB related programs, and assist with the final DNA annotation review for submission to PhagesDB.

Currently, this repo has a datascraper meant to get data on HHPRED names to train an algorithm to interpret raw HHPRED results into a given function on the SEA-PHAGES official function list (https://seaphages.org/blog/official-function-list/). The algorithm itself is yet to be completed (in part because the methodology of pulling data is incredibly slow and needs to be refined or fully changed) and is not uploaded. Additionally there are other parts to the tool that have been largely completed, however in Java, and as such will be rewritten in Python before uploading.
